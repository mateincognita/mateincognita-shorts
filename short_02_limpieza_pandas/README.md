
# Short 02 — Limpieza de Datos con Pandas

Canal: [MateIncognita](https://www.youtube.com/@mateincognita)

##
Antes de analizar cualquier dato, necesitas limpiarlo.
Datos sucios = reportes incorrectos.

## 3 pasos de limpieza básica

1. **Eliminar duplicados** — `drop_duplicates()`
2. **Rellenar valores nulos** — `fillna()`
3. **Corregir tipos de datos** — `astype()`
